Each game has its own unique `game_id` generated by the backend.

**Open the main index page**  

```
http://127.0.0.1:5000/
```
Here the **Moderator** can log in using a password and create a new game session.

**Click â€œNew Game.â€**  
This triggers the `/create_game` route, which:
- Creates a new record in the database

  - Player 1  
  - Player 2  
  - Moderator view
